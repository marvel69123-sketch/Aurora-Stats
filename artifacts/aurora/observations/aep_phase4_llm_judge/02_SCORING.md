# AEP Phase 4 — Scoring

## Dimensions (0–10)

| Field | Meaning |
|-------|---------|
| `understanding_score` | Intent / term / pronoun grasp |
| `utility_score` | Useful, non-loop reply length/content |
| `continuity_score` | Fixture/context reuse after sport turns |
| `credibility_score` | No invention on INVALID; honest limits |
| `naturalness_score` | Human, non-crumb replies |
| `clarity_score` | Structure / readable length |
| `overall_score` | Weighted blend |

## Bands

| Score | Label |
|------:|-------|
| 9–10 | Excelente |
| 7–8 | Boa |
| 5–6 | Aceitável |
| 0–4 | Ruim |

## Compact output aliases

`overall`, `understanding`, `continuity`, `utility`, `credibility`, `naturalness`, `clarity`
