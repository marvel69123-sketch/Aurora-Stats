# Replit — ponte para o procedimento único

O guia completo de install + deploy está em:

**→ [DEPLOY.md](./DEPLOY.md)**

Resumo no Replit Shell:

```bash
cd ~/workspace
git pull origin main
pnpm run deploy
# depois: Republish + Ctrl+Shift+R
```

Fonte de verdade: **GitHub `main`**. Não dependa de commits automáticos do Replit Agent.
